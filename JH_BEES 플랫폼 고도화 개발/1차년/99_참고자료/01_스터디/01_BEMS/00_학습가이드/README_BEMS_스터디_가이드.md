# BEMS 스터디 가이드

## 1. 목적

건물 자율운전 플랫폼 기획에 필요한 BEMS 기본 개념을 학습하고, 추후 요구사항 정의에 활용할 수 있도록 자료를 분류한다.

현재 원본 PDF는 외부 드라이브 접근 제한으로 복사하지 못했다. 원본을 전달받으면 아래 분류에 맞춰 저장하고, 각 문서의 핵심 내용을 보완한다.

## 2. 먼저 이해할 질문

1. BEMS는 어떤 데이터를 수집하고 어떤 기능을 제공해야 하는가?
2. 관제점은 어떤 기준으로 선정하고 어떤 규칙으로 이름을 붙이는가?
3. 태그와 데이터베이스는 향후 디지털 트윈 및 자율운전 기능과 어떻게 연결되는가?
4. 설비 효율은 어떤 지표로 판단하며, 에너지 절감 성과는 어떻게 측정하는가?
5. 설치 확인과 운영 시 준수해야 할 기준은 무엇인가?

## 6. 원본 수신 후 정리 규칙

1. 원본 PDF는 수정하지 않고 각 분류 폴더에 보관한다.
2. 동일 자료를 여러 폴더에 복사하지 않는다.
3. 복수 주제에 해당하는 자료는 이 가이드에서 관련 폴더를 참조한다.
4. 문서별 요약본에는 작성일, 작성자, 원본 파일명을 기록한다.
5. 표준 및 운영 규정은 적용 시점의 최신 여부를 별도로 확인한다.
